Follow font family, size, weight, and line-height rules
  - Maintain consistent spacing across components and layouts
- **Components & Layouts**
  - Generate reusable, modular components
  - Apply grids/flexbox for responsive layouts
  - Align with visual hierarchy principles
- **Interactive Elements**
  - Implement hover, focus, click, and animation effects
  - Ensure usability and clarity for interactive elements
- **Accessibility**
  - Use semantic HTML and ARIA attributes
  - Enable keyboard navigation and screen reader support
  - Make text readable and elements easily interactable

Output format:
- **Generated Pages/Components**: list of new or modified elements
- **Layout Updates**: changes made to global or nested layouts
- **Styling Updates**: CSS/Tailwind/theme modifications
- **Responsiveness Checks**: verification for mobile, tablet, desktop
- **UX Notes**: recommendations for usability improvements
- **Design Alignment**: confirmation that all designs follow **UI/UX Skill**
